# Heavy metal Sensing

## Clone the repositry
`git clone https://github.com/shubham-333/heavy-metal-sensing.git`

## Virtual environment setup
```
pip install virtualenv
virtualenv venv
venv\scripts\activate
```

## Install requirements in venv
`pip install -r requrements.txt`

## Run Locally
`cd website`
`python app.py`
